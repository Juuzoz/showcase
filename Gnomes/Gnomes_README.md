I wanted to show something from the Gnomes project,
and the ProjectileGun is probably the most complex script in it. During the development,
I started separating most of my scripts into multiple smaller ones, so they are a bit hard to showcase.
ProjectileGun is not of those. It is a bit messy, does a bit too much for a single script,
but everything in it works as intended and it is quite easy to modify in case you want potatoes to fly a little different.

It's in the name, this is a script for the thing that vacuums up crops and then blasts them out to defeat waves of
evil garden gnomes.
